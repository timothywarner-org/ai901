# Lesson 15: Build a Computer Vision and Image-Generation Application

**Runtime:** 30 min | **Exam objectives:** 2.3.1 -- 2.3.3

## Learning objectives

- Interpret visual input in prompts by using a deployed multimodal model.
- Create new visual outputs by using generative models.
- Build a lightweight application that includes vision capabilities.

## Demo

Python app that performs both directions: send an image to a multimodal Foundry model and ask for a description, then generate an image from a text prompt. Code lives in [`demo/`](./demo/).
