# Lesson 12: Build a Lightweight Client Application for an Agent

**Runtime:** 30 min | **Exam objectives:** 2.1.5

## Learning objectives

- Create a lightweight client application for an agent.
- Connect a client application to a Foundry agent endpoint by using the Foundry SDK.
- Send user messages to an agent and handle multi-turn conversation state from the client application.

## Demo

Python client that calls the agent built in lesson 11 -- threads, multi-turn state, and rendering tool calls. Code lives in [`demo/`](./demo/).
