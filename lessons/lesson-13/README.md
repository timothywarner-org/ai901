# Lesson 13: Build a Text Analysis Application with Foundry

**Runtime:** 30 min | **Exam objectives:** 2.2.1

## Learning objectives

- Build a lightweight application that includes text analysis.
- Call Azure AI Language capabilities from a client application by using the Foundry SDK.
- Extract entities, key phrases, sentiment, and summaries from unstructured text and display the results in a client UI.

## Demo

Python text-analysis app -- entities, key phrases, sentiment, summarization on a sample document, rendered to a minimal CLI or Streamlit UI. Code lives in [`demo/`](./demo/).
