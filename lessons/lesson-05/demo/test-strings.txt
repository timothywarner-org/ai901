[7] FOUR-WAY TRADEOFF MATRIX
    Read aloud from a second monitor or notepad -- NOT pasted into any portal.
    This is the close-out visual for the classic-vs-generative NLP comparison.
================================================================================

Dimension     | Classic Azure AI Language        | Generative gpt-4.1-mini
--------------+----------------------------------+--------------------------------
Cost          | Per call, predictable pennies    | Per token, scales with I/O
Latency       | Single API call, ~100 ms         | Token-by-token, ~1+ second
Auditability  | Versioned API + fixed schema     | Non-deterministic;
              |                                  | prompt + model = contract
Flexibility   | One named task per endpoint      | Any task you can describe

Rule of thumb:  named technique  ->  prebuilt Azure AI Language
                open-ended / multi-step / reasoning  ->  generative model
                in doubt  ->  both, on different parts of the workload
