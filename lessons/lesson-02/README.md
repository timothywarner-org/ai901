# Lesson 2: How Generative AI Models Work and How to Choose Them

**Runtime:** 30 min | **Exam objectives:** 1.2.1 -- 1.2.3

## Learning objectives

- Describe how generative AI models work.
- Identify an appropriate AI model, based on capabilities.
- Identify appropriate model deployment options and configuration parameters.

## Demo

Walk the Microsoft Foundry model catalog: filter by task, context length, and cost; compare deployment types (Standard, Provisioned, Batch) and key parameters (temperature, top_p, max tokens).
